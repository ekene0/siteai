Introduction:

The data set imported, contains 1000 data points; the objective here is to apply the data science process to the data set developed by the solar monitoring data acquision system. The first step involved cleaning the dataset, i.e making the data uniform and analysis ready. The next step is to visualize the data in order to have a better understanding of the relationship between the variables. Also the data was trained with a linear regression algorithm and a prediction coefficient was established; finalized with a performance score for the algorithm.
